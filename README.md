# Middle Python Backend Roadmap 2026

Интерактивный roadmap для изучения **Python backend-разработки** — от базового Python до уровня **Middle Python Backend Developer**.

Проект помогает не просто читать список тем, а **проходить путь по неделям**, отмечать прогресс, раскрывать задания по дням, смотреть проекты недели, ресурсы и вопросы для самопроверки.

---

## О проекте

Этот roadmap разбит на **4 фазы** и **18 недель** обучения:

- **Фаза 1 — Python Core**
- **Фаза 2 — Web Framework + Databases**
- **Фаза 3 — Tests + DevOps**
- **Фаза 4 — Architecture + Portfolio**

Внутри каждой недели есть:

- план **по дням**;
- список **ключевых тем**;
- **проект недели**;
- **ресурсы** для изучения;
- **вопросы для самопроверки**.

---

## Что умеет приложение

- раскрытие roadmap по неделям;
- просмотр плана **по дням**;
- отслеживание статуса тем:
  - не начато;
  - в процессе;
  - готово;
- отметка завершённых недель;
- отображение общего прогресса;
- блок с проектом недели;
- подборка учебных ресурсов;
- вопросы для самопроверки;
- поиск по содержимому roadmap;
- сохранение прогресса в браузере.

## Как выглядит логика приложения

Приложение построено вокруг массива `roadmap`, где каждая фаза содержит:

- название фазы;
- цвет фазы;
- недели;
- дни;
- темы;
- проект недели;
- ресурсы;
- вопросы.

Это делает проект удобным для расширения: можно добавлять новые недели, менять структуру, адаптировать roadmap под другой стек или под другой уровень.

## Запуск локально

### 1. Клонируй репозиторий

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
cd YOUR_REPOSITORY
```

### 2. Установи зависимости

```bash
npm install
```

### 3. Запусти проект

Если у тебя **Vite**:

```bash
npm run dev
```

Если у тебя **Create React App**:

```bash
npm start
```

---

## Production build

```bash
npm run build
```

## Как пользоваться

- нажми на нужную неделю, чтобы раскрыть детали;
- переключай вкладки:
  - **По дням**
  - **Темы**
  - **Проект**
  - **Ресурсы**
  - **Вопросы**
- кликай на тему, чтобы менять её статус;
- отмечай завершённые недели;
- следи за общим прогрессом в header.
